# UI

You need to setup a simple UI to list notes. The UI should show the notes in
chronological order, with the most recent notes at the top of the file.

At the top of this list, should be a textarea for users to input new notes.
When they press enter or click on the submit button, the note should be
submitted to the backend. Shift+Enter should add new lines without submitting.

I recommend you use next.js pages and server actions. The page component can
do the data loading, and the server action can support the submission of new
notes.

Keep styling basic and minimal for now. Focus on core functionality.
