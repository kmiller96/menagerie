# Single Page Demo

Demo to showcase how you can use Vite to bundle into a single file.

## Quickstart

```bash
pnpm build
```

Find the built artifact at `./dist/index.html`.

## Why?

It can be useful to create a single, self-contained HTML "binary" that can be
shared with users and run entirely offline. But you still want all the tooling
and structure from a standard react project.
